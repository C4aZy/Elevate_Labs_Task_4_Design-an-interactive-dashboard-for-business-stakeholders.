# 📊 Dashboard Design – Car Sales Analysis (Tableau)

## 📌 Project Overview

This project focuses on designing an **interactive dashboard** using car sales data. The goal is to help business stakeholders understand sales trends, growth patterns, and shifts in fuel types.

The dashboard is built using **Tableau** and provides insights into monthly and yearly car sales, fuel distribution, and environmental impact.

---

## 🎯 Objectives

* Analyze car sales trends over time
* Compare different fuel types (Diesel, Hybrid, Electric)
* Study year-over-year (YoY) growth
* Understand CO₂ emission trends
* Build an interactive dashboard for decision-making

---

## 🛠️ Tools Used

* Tableau (for dashboard creation)
* Microsoft Excel (dataset)

---

## 📂 Dataset

* Dataset: Norway New Car Sales by Month
* Features include:

  * Year, Month
  * Quantity (Total Sales)
  * Quantity_YoY (Growth)
  * Import, Used
  * Fuel types (Diesel, Hybrid, Electric)
  * CO₂ emissions

---

## 📊 Dashboard Features

* 📈 **Sales Trend Analysis** (Year & Month-wise)
* 🔥 **Fuel Type Comparison** (Diesel vs Hybrid vs Electric)
* 🌍 **CO₂ Emission Trend**
* 📅 **Monthly Heatmap** for seasonal patterns
* 🎛️ **Filters** (Year, Month) for interactivity
* 📌 **KPI Cards** (Total Sales, Electric Sales, Avg CO₂)

---

## 💡 Key Insights

* Car sales show seasonal variation, peaking mid-year
* Electric vehicle adoption is increasing significantly
* Diesel vehicle share is gradually declining
* CO₂ emissions are decreasing over time, indicating a shift toward cleaner energy
